# 설치 모듈

- npm i express (익스프레스 사용)

  ```bash
  npm i express
  ```
- npm i dotenv (env 사용)

  ```bash
  npm i dotenv
  ```  
- npm i typescript -D (타입 스크립트 사용)

  ```bash
  npm i typescript -D
  ```
- npm i @types/node @types/express @types/dotenv -D

  ```bash
  npm i @types/node @types/express @types/dotenv -D
  ```  
- npm i ts-node-dev -D (자동으로 변경된 파일 실행)

  ```bash
  npm i ts-node-dev -D
  ```
